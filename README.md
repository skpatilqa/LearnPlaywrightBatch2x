# JavaScript Identifier Rules and VS Code Shortcuts

## JavaScript Identifier Rules

### 1. Allowed Characters
- Letters (a–z, A–Z), digits (0–9), underscores (`_`), and dollar signs (`$`)
- Examples:
  - `name`
  - `userName`
  - `item1`
  - `_count`
  - `$price`

### 2. Cannot Start with a Digit
- Valid: `var1`
- Invalid: `1var`

### 3. Case-Sensitive
- `myVar`, `MyVar`, and `MYVAR` are three different identifiers

### 4. Cannot Use Reserved Keywords
- Invalid examples: `let`, `class`, `return`, `function`, `if`

### 5. No Spaces or Special Characters
- Invalid examples:
  - `my name`
  - `user@name`
  - `price#`

### 6. Unicode Letters Allowed
- Examples:
  - `变量`
  - `résumé`

## VS Code Shortcuts for Windows

| Action | Shortcut |
|--------|----------|
| Multi-line editing (multiple cursors) | `Ctrl + Alt + ↑ / ↓` |
| Comment / Uncomment multiple lines | `Ctrl + /` |
| Block comment (`/* text */`) | `Shift + Alt + A` |
